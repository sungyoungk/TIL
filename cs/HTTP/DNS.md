## 🔎 DNS, 도메인 네임 시스템(Domain Name System)
-  도메인 명을 등록하고 IP 주소로 변환한다
-  이렇게 도메인 명을 IP 주소로 변환되면 기억하기도 쉽고, 중간에 변경되어도 도메인명과 IP가 다시 연결되기 때문에 문제가 해결된다
